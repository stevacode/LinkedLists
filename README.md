# Linked Lists - C++ Practice

A C++ project for practicing and implementing linked list data structures.

## 📋 Description

This project is focused on learning and implementing various linked list operations in C++. It serves as a practice ground for understanding fundamental data structures and pointer manipulation.

## 🛠️ Technologies

- **Language**: C++20
- **Build System**: CMake 3.28.3
- **Generator**: Ninja
- **Toolchain**: GNU
- **IDE**: Visual Studio 2026

## 📁 Project Structure

```
LinkedLists/
├── CMakeLists.txt      # CMake build configuration
├── CMakePresets.json   # CMake presets
├── LinkedLists.h       # Header file
├── LinkedLists.cpp     # Main implementation
├── out/                # Build output directory
└── README.md           # Project documentation
```

## 🚀 Getting Started

### Prerequisites

- CMake (minimum version 3.10)
- C++20 compatible compiler (GNU toolchain)
- Ninja build system

### Building the Project

```bash
# Configure the project
cmake --preset=default

# Build
cmake --build out/build/x64-debug
```

### Running the Application

```bash
./out/build/x64-debug/LinkedLists
```
